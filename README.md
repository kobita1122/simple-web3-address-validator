# Simple Web3 Address Validator

This is a beginner-friendly blockchain project.

This project helps you:
- Enter a wallet address
- Check if the address is valid
- Learn how blockchain addresses work
- See validation result on screen

This project uses:
- HTML
- JavaScript
- Simple validation logic

## Features
- Input wallet address
- One-click validation
- Show valid or invalid result
- Clean and simple layout

## Requirements
- Internet connection
- Basic JavaScript knowledge

## How to Run
1. Download this repository
2. Open `index.html` in browser
3. Enter a wallet address
4. Click "Validate"
5. See result on screen

## File Structure
- index.html -> Main webpage
- app.js -> App logic
- validate.js -> Address validation
- ui.js -> Screen updates
- config.js -> App settings
- message.js -> Result messages
- format.js -> Format output
- logger.js -> Debug logs
- helper.js -> Helper functions
- utils.js -> Extra tools

## Learning Goal
This project helps beginners learn:
- What a wallet address is
- How basic validation works
- How frontend logic is structured
